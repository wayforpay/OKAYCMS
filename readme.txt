Инструкция по установке:-------------------------- Скопируйте новые папки и файлы с сохранением пути.
- Для того, чтобы автоматически менялся статус заказа на Оплачен, в личном кабинете WayForPay в настройках в разделе Настройки магазина->Уведомления на service URL, напишите:http(s)://ваш_сайт.ru/payment/wayforpay/callback.php--------------------------Примечание: Инструкция предполагает, что дополнение ставится на чистую OkayCMS со стандартным шаблоном. Для магазина с измененным кодом, процесс установки может отличаться.
